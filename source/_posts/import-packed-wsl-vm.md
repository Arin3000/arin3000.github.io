## Import .tar

先切换目录，在 Windows 环境下的命令提示符位置处理。

```bash
C:\Users\Arin>wsl --import UbuntuGVMHR D:\wsl D:\wsl\ubuntu2004.tar
```

切换用户：

```bash
sudo -u joe -i
```
